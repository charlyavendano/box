# Box Express de El Salvador

Modern, responsive website for a courier/shipping company operating between San Miguel, El Salvador, and Houston, Texas.

## Tech Stack

- Angular 18 (Standalone Components)
- Tailwind CSS
- Lucide Angular (Icons)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open your browser and navigate to `http://localhost:4200`

## Project Structure

- `src/app/components/` - Reusable components (Navigation, Footer, Services)
- `src/app/pages/` - Page components (Home, Services, Quote, Trips, Register)
- `src/assets/` - Static assets including logo.png

## Features

- **Navigation Bar**: Logo on left, navigation links on right
- **Hero Section**: Auto-playing image carousel
- **Services Section**: Cards for San Miguel and Houston locations
- **Quote Calculator**: Real-time pricing calculator with invoice preview
- **Trips Calendar**: Interactive calendar showing February 2024 trip schedule
- **Registration Form**: User registration with validation

## Logo

Replace `src/assets/logo.png` with your actual logo file. The logo should be a PNG image that will be displayed in the navigation bar.

## Color Palette

- **Primary (Navy)**: `#1e3a5f` - Used for navbar, footer, and primary headings
- **Accent (Cyan)**: `#00bcd4` - Used for CTAs, icons, and hover effects
- **Background**: White and light gray/blue tints
